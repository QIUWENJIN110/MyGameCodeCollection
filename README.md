# MyGameCodeCollection

This repository contains two classic game projects developed in C++ with Qt 5.14.2, as part of my application portfolio for postgraduate study in Computer Game Technology.

---

## Projects

- **Game2048**: A C++/Qt implementation of the 2048 puzzle game, focusing on grid logic, UI rendering, and input handling.
  - See [Game2048/README.md](./Game2048/README.md) for details.

- **SnakeGame**: A classic snake game built with Qt, demonstrating 2D rendering, game loop design, and collision detection.
  - See [SnakeGame/README.md](./SnakeGame/README.md) for details.

---

## Build & Run

1.  Open the project `.pro` file in Qt Creator (Qt 5.14.2 or later).
2.  Configure and build for your target platform.
3.  Run the application.
